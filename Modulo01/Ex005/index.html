Principais formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag <b> (Não semâmtica).

Nesta frase, tenho um termo em destaque usando a tag <STRONG>(Semâmtica)

Itálico / Ênfase

Nesta frase, temos um termo em italico usando a tag <i> (Não semâmtica)
Pode se usar o atalho CTRL+SHIFT+P=Envelope com abreviatura, para envolopar uma frase dentro de uma <tag>

Nesta frase, temos um termo em ênfase usando a tag <em>(Semâmtica)

Texto marcado

Podemos criar também um texto marcado usando a tag <mark>

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno nesse páragrafo usando as tags <big> e <small>.

Texto deletado

Podemos marcar um texto como excluido para indicar que ele deve ser lido, mas não considerado usando a tag <del>.

Texto inserido

Podemos marcar um texto como inserido para dar enfase e indicar que que ele foi adicionado depois usando a tag <ins>(Semantica)

Existe também o sublinhado com a tag <u> (Não semantica)

Texto sobrescrito

Para inserir coisas do tipo x20+3 usando a tag <sup>

Texto subscrito

Para inserir coisas do tipo H20 usando a tag <sub>

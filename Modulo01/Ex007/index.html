Listas ordenadas

Para criar listas ordenadas basta usar as tags <ol> e <li>
É possivel usar o parametro "type" para definir o padrão de ordem da lista ex.: numeros ou letras para
Também é possivel definifir em qual numero ou letra começara a lista com o parametro "start"
